# Traveling Salesman Problem - Genetic Algorithm

---

## Requirements

- Node
- Python 3.x
- Pip

## Run development

### Server

- `pip install -r requirements.txt`
- `python3 __init__.py`
- Now server is running in `http://0.0.0.0:5000/`

### Client

- `npm install`
- `yarn serve`
- Now client is running in `http://localhost:8080/`
